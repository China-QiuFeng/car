# 汽车保养维修信息管理系统

### 介绍
一款基于FastAdmin+ThinkPHP和uniapp开发的汽车保养信息管理系统，旨在助力汽车企业保养全流程精细化、数字化管理，主要功能：客户、产品、服务、收费、优惠券，适用于：汽车维修、汽车保养、大型洗车店行业等。提供全部无加密服务端和前端源代码，支持私有化部署
